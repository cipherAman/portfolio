## ✨ Cutting-Edge Key Features

### 🐳 1. Interactive 3D Docker Artifact Card
* **Apple TV-style 3D Tilt:** Smooth, hardware-accelerated 3D tilt rotation tracking mouse coordinates with precise parallax perspective.
* **Specular Radial Shine:** A dynamic light overlay that tracks mouse movement, creating realistic neon-red obsidian glass specular glints.
* **Hover-Activated 360° Y-Axis Spin:** Triggers an infinite 3D spin on hover for an immersive, performance-optimized interaction.

### 🌌 2. 3D Particle Constellation (Plexus) Background
* **HTML5 Canvas Engine:** High-performance, custom-coded interactive background drawing live particle nodes.
* **Mouse Interactivity & Gravity:** Nodes dynamically shift parallax depth, tracking cursor movement and building connective neural constellation links with smooth drag physics.

### ⚡ 3. Flowing Technical Skills Engine
* **Flowing Shimmer Progress Bars:** Sleek gradient animation pipelines representing technical expertise level.
* **Subtle Micro-Animations:** Breathing interactive cards and radial lighting indicators.

### ✉️ 4. Serverless Recruiter Lead Capture
* **Instant Email Forwarding:** Secured Web3Forms integration that forwards messages directly to `aman.enthusiast@gmail.com`.
* **Zero Client-Side Keys:** Complete protection using server-isolated environment variables in `.env.local` to prevent scrapers.

### 📱 5. Fluid Mobile Responsiveness
* **Header Clearance Engine:** React state-based responsive layout calculations to perfectly center typography and prevent overlap with the fixed top navigation bar.

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/cipheraman/portfolio.git
cd portfolio
```

### 2. Install dependencies
```bash
npm install
```

### 3. Add Environment Variables
Create a `.env.local` file in the root directory:
```env
NEXT_PUBLIC_FORM_ENDPOINT="https://api.web3forms.com/submit"
NEXT_PUBLIC_WEB3FORMS_KEY="YOUR_WEB3FORMS_KEY_HERE"
```

### 4. Run the development server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to view your premium portfolio live!
